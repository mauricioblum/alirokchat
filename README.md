# Alirok Chat

A React chat application built with modern technologies.

## Features

- Responsive
- Live chat with JSON server
- React Hooks
- Context API

## Run Locally

Clone the project and cd into the directory.

Install dependencies

```bash
  npm install

  // or

  yarn install
```

Run with

```bash
  npm run start

  // or

  yarn start
```
